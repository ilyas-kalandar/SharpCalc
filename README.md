# SharpCalc
### Tiny calculator, created with C# & .NET
Currently support +, -, /, * and ()

## How to run?

#### Clone repo

```bash
git clone https://github.com/ilyas-kalandar/SharpCalc
```

#### Change directory
```bash
cd SharpCalc
```
#### Run
```bash
dotnet run 
```
## Example
```bash
SharpCalc>> (10 * 3) + (6 * 2) + 8
Ans: 50
SharpCalc>> 2 + 7 * 3
Ans: 23
SharpCalc>>
```
