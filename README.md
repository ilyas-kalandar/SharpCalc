# SharpCalc
### Tiny calculator, created with C# & .NET

### Currently support +, -, /, * and ()

## How to run?
<code>cd SharpCalc</code><br>
<code>dotnet run</code>

## Example
<code>SharpCalc>> (10 * 3) + (6 * 2) + 8</code><br/>
<code>Ans: 50</code><br/>
<code>SharpCalc>> 2 + 7 * 3</code><br/>
<code>Ans: 23</code><br/>
